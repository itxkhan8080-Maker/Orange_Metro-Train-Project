# Orange Metro Line Service

## 📌 Overview

A console-based C++ application that simulates a metro ticketing system for the Orange Metro Line in Lahore. Users can recharge their metro card or buy a token by selecting a destination station.

---

## 🎯 Objectives

- Practice Object-Oriented Programming in C++
- Implement class design with real-world entities
- Simulate a basic ticketing and recharge workflow

---

## 🚀 Features

- Metro card recharge with note validation
- Token purchase with station selection
- Displays station names and prices
- Route slip confirmation system

---

## 🛠️ Technologies Used

- C++
- OOP (Classes & Objects)
- Standard I/O (`iostream`, `string`)

---

```
oop-final-project.cpp
```

## ⚙️ How to Run

1. Clone the repository
   ```
   git clone <repository-url>
   ```

2. Compile the program
   ```
   g++ oop-final-project.cpp
   ```

3. Run the executable
   ```
   ./metro
   ```

---

## 💻 Usage

- Launch the program
- Choose **1** to recharge your metro card or **2** to buy a token
- If buying a token, select your destination station from the list
- Follow the on-screen prompts to complete the transaction
